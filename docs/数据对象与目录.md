# 五个核心对象与目录

## PathogenProfile

位于 `profiles/<profile_id>/`，包含人工种子、词典、查询组、来源注册表、搜索/模型/编辑策略和 provenance。人工审核后才能更新生产版本。

## ScholarlyWork

多文献接口记录合并后的实体。主键 `work_id` 保持稳定；标识符、标题、摘要、书目信息、实体、关系、字段来源、冲突和 AI 分析分开保存。

## NewsArticle

一篇官方通报或媒体文章。保存规范 URL、来源等级、原文、证据句、指纹、规则决策、AI 分析和 `event_id`。

## PublicHealthEvent

由一篇或多篇 NewsArticle 聚合，使用稳定 `event_id`、`event_version` 和 `change_history` 表示跨日更新。

## DailyIssue

一天的最终发布快照，只引用文献和事件 ID，并保存统计、栏目、来源健康、质量说明、模型审计和输出路径。
