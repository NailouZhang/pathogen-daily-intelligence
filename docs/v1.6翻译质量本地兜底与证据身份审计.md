# v1.6 翻译质量、本地兜底与证据身份审计

## 修复对象

真实网页出现中文标题缺失、宋病毒/汉塔病毒、明显汉坦病毒疫情、汉坦病毒肆虐游轮、中英文混排、Google News 聚合模板被翻译成摘要，以及翻译通过但模型尝试链显示失败等问题。

## 新流程

1. 删除聚合模板和空摘要占位。
2. 远程模型严格执行翻译任务，不再同时承担自由总结。
3. 结构、数字、占位符、术语、中文比例、残留英文、否定和不确定性校验。
4. 以 MarianMT 标题译文作为独立参考信号。
5. GitHub Models、Gemini、Groq 全部失败或译文被拒绝后，由 MarianMT 翻译标题和已获取正文。
6. 没有正文时只翻译标题，卡片明确提示未抓获正文。
7. 翻译审计与内容分析审计分开。
8. DOI 仅出现在 PDF 中不再足以证明 PDF 身份；题名或作者必须相容。

## 性能

MarianMT 仅在真实任务启用，模型由 GitHub Actions 缓存。第一次运行需要下载模型，后续运行复用缓存。回译模型默认关闭。
